3. Variables

Variables produce a value, like fps, xyz, biome, etc
Syntax: {<variable_name> [flags]}
Examples: {fps} , {x} , {biome} , {ping}
We're saving flags for the next part

Reference: All the variables available. (You can safely ignore the Number Value and Boolean Value columns for now)

Instructions

Let's create a profile that shows our fps, coordinates, and biome. {bx} , {by} , {bz} are the player's block coordinates.

Profile File

profile1.txt
fps: {fps}
xyz: {bx} {by} {bz}
biome: {biome}

Result In-Game
